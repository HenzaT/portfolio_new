Projects

Gimme a Song

song idea app for musicians

A self-directed project built to help musicians overcome writer's block by generating random song elements. Built using Ruby on Rails, PostgreSQL, and JavaScript. The front end communicates with the back end via the Fetch API for dynamic data handling. I also integrated Devise for user authentication and implemented CSRF protection to ensure secure user interactions.

React Weather App

Flask & React Weather app

I built this full-stack application to solidify my understanding of React and TypeScript. The app integrates a Flask backend and consumes external APIs: OpenWeather for real-time weather data and the Claude AI API from Anthropic for activity suggestions. My focus was on creating smooth UX, while deepening my knowledge of asynchronous data handling and API integration.

About Me

Background

My passion for tech started when I began writing and recording my own music. I was fascinated by how the DAW (digital audio workstation) worked and I loved the creative designs of the sound plugins. Wanting to learn more about how they were built, I enrolled in a web development course and have been building and learning ever since.

About

I'm a software engineer with a background in education, passionate about building intuitive, user-focused products. I love bringing ideas to life: whether it's a website, a lesson, or a song. I have experience building responsive websites, building servers, and b2b IT support. I'm currently consolidating my knowledge of Perl, Linux and scripting languages.

Interests

Outside of coding and tech, I write and produce my own music as well as play bass guitar in a band. I also love to climb and travel - I'm half Japanese, so I'm lucky enough to go to Japan at least once a year to visit family (and stock up on tea!)
