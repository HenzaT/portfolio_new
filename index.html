Projects

Gimme a Song!

song idea app for musicians

I built this website in response to a common problem I and many other musicians have faced: how to start a song. My website randomly gives you elements of a song to start off with, for example the time signature and key. This was my first solo project which I started during Le Wagon, and I heavily utilised PostgreSQL and the fetch API through JavaScript.

Morning Burrito

mobile app for creative activities

My final team project at Le Wagon. This app is intended to help people to be more creative throughout the day by completing activities like writing a Haiku, or drawing something inspired by found sounds. I utilised JavaScript (Rails' Stimulus framework) heavily to improve the user experience on the preferences and dashboard pages.

Blue Horizon

rental market space for yachts

My first team project at Le Wagon. We were tasked with building an airbnb style website and so naturally we settled on renting yachts. I focused mainly on the frontend, working with Bootstrap and JavaScript to style multiple pages according to the lead designer's designs. Using PostgreSQL, I also built the backend of the dashboard page.

Portfolio

personal portfolio for projects

Although admittedly this is not a product, I used this portfolio as an opportunity to consolidate my HTML, CSS and JavaScript skills. I focused on design and animations, particularly JavaScript DOM manipulation and CSS design. I also wanted to use TypeScript, having completed a course on it recently. I hope to apply what I have learned here to past projects.
